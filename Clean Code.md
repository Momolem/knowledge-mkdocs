# CleanCode
Clean Code sind Praktiken, welche die Qualität des Codes durch BestPractices verbessern sollen.

Die Prinzipien basieren Hauptsächlich auf dem Buch [Clean Code (Book)](Clean%20Code%20(Book).md) von [Robert C. Martin](CleanCode/Robert%20C.%20Martin.md).

Ein Praktischer Ansatz um Clean Code zu meistern ist der [Clean Code Developer](CleanCode/Clean%20Code%20Developer.md)
